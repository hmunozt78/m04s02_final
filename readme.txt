Sesion 02
Ejercicio - Final (Drill)

Para Clonar el repositorio, ejecute la siguiente instruccion:

git clone https://github.com/hmunozt78/m04s02_final.git


Se solicita la creacion de una clase de tipo persona, con los siguientes
parametros:

nombre - str
apellido - str
sexo - str
edad - int
estatura - float
peso - int

Se crearon Getter y Setter para cada elemento, realizando las pruebas
correspondientes segun lo indicado en el documento.

Se agrega estilo de colores a traves de las instrucciones \033, que tiene la 
siguiente forma de ejecutar

\033["codigo estilo";"codigo texto";"codigo fondo"m

\033[ - Inicia la instruccion
"codigo estilo" - va de 0 a 7, segun estilos desde sin estilo, hasta tachado
"codigo texto" - va de 30 a 37
"codigo fondo" - igual que anterior
m - cierra la instruccion
